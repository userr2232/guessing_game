# Guessing game

## Setup

If you are on macOS:

```curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh```

and then reload the terminal.

```git clone https://github.com/userr2232/guessing_game.git```

## Running the code

to run the project, cd into the project

```cd guessing_game```

and then

```cargo run src/main.rs```
